# AscendTech ROV Code[![GoDoc](https://godoc.org/github.com/AscendTech4H/AscendTechROV?status.svg)](https://godoc.org/github.com/AscendTech4H/AscendTechROV)

For Arch Linux, run: ./archsetup.sh
